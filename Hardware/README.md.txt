# Hardware Folder
This folder contains information about Muninn's hardware setup.

## Components
1. **Servos**:
   - MG90S Metal Gear Servos
   - Used for neck tilt, rotation, and beak control.

2. **LEDs**:
   - WS2812B Addressable LEDs
   - Provide blinking eye effects.

3. **Microcontrollers**:
   - Arduino Uno for servo and LED control.
   - Raspberry Pi 4 model B for AI and image processing.
